# spec 目录下的文件清单

本目录包含县级养老补贴发放系统的需求与设计文档，供开发、测试与验收使用。

| 文件名 | 说明 |
|--------|------|
| **县级养老补贴发放系统用户需求说明书（完整版）.md** | 用户需求说明书：项目背景、功能需求（基础信息、补贴计算、发放管理、账户与财务、报表、权限）、交付物 |
| **概要设计文档.md** | 概要设计：角色定义、系统架构、审批流程、功能模块、数据库设计、菜单结构、技术实现要点、实施计划 |
| **管理使用流程.md** | 管理使用流程：补贴人员登记（2级/3级审核）、待遇人员管理、待遇发放、支付计划复核与审批等 |
| **快速启动指南.md** | 开发与运行：环境准备、项目结构、阶段一至十、开发规范、常见问题 |
| **被征地参保补贴重构设计.md** | 被征地参保补贴独立模块的详细设计（登记、申领、支付计划、综合查询） |
| **现有系统分析.md** | 现有系统分析 |
| **重构规划总结.md** | 重构规划总结 |
| **sequential-thinking-analysis.json** | 顺序思维分析数据（如有） |
| **sequential-thinking-analysis.lock** | 顺序思维分析锁文件（如有） |

## 需求与实现对照

- **功能需求** 以《用户需求说明书》和《概要设计文档》为准。
- **审批流程** 以《管理使用流程》和《概要设计文档》中的状态机、角色职责为准。
- **被征地参保补贴** 以《被征地参保补贴重构设计》和《概要设计文档》4.2.6 节为准。

测试与开发时请据此对照实现，列出不符合项并逐个完善。
