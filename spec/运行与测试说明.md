## 二、运行当前项目

### 1. 环境要求

- JDK 17+
- Maven 3.6+
- MySQL 8.0（库与账号按项目配置）
- Node.js 14+（前端）
- Redis（可选）

### 2. 后端

**若使用私有 Nexus 且无法连接**，可先用项目内的阿里云配置编译并安装依赖，再启动：

```powershell
# 1. 编译并安装所有模块到本地仓库（仅首次或依赖变更后需要）
mvn -s settings-aliyun.xml install -DskipTests

# 2. 设置环境变量（或使用 .env 中的值），再启动
$env:MYSQL_HOST="www.htmisoft.net"; $env:MYSQL_PORT="36522"; $env:MYSQL_DB="lfpm"; $env:MYSQL_USER="root"; $env:MYSQL_PWD="你的密码"; $env:REDIS_HOST="www.htmisoft.net"; $env:REDIS_PWD="你的密码"; $env:PROFILE_PATH="e:\langfangsb\lfsb\uploadPath"
mvn -s settings-aliyun.xml spring-boot:run -pl ruoyi-admin
```

后端默认端口 **8087**，上下文路径 **/api**，即：`http://localhost:8087/api`。

需确保 MySQL、Redis 可连接（当前 `.env` 指向的数据库若在本机不可达，会报 `Communications link failure`，请改为本机或可访问的地址）。

### 3. 前端

```bash
cd ruoyi-ui
npm install
npm run dev
```

访问前端地址（如 http://localhost:80 或控制台提示的端口）。

### 4. Cloud Agents 环境（可选）

若使用 Cursor Cloud Agents，项目已配置 `.cursor/environment.json`，会执行：

- `install`：`mvn compile -DskipTests` + `cd ruoyi-ui && npm install`
- `terminals`：前端 `npm run dev`、后端 `mvn spring-boot:run -pl ruoyi-admin`

## 三、JUnit 测试

### 1. 运行测试

在项目根目录执行（需能解析 Maven 依赖）：

```bash
# 仅 langfang-shebao 模块
mvn -pl langfang-shebao test

# 全项目
mvn test
```

### 2. 已编写的测试

| 测试类 | 说明 |
|--------|------|
| **SubsidyCalculationServiceImplTest** | 补贴计算：补贴年限（16～45→3年、46～50→5年、51～59→年龄-45、60+→15年）、年龄计算、补贴金额、年平均工资默认值 |
| **SubsidyTypeEnumTest** | 补贴类型：按编码/表名获取枚举，以及需求规定的 5 种类型（含教龄补助） |

### 3. 依赖说明

`langfang-shebao/pom.xml` 已增加 `spring-boot-starter-test`（scope=test），用于 JUnit 5 与 Mockito。若本地 Maven 使用私有仓库且无法连接，可临时改用阿里云等镜像后再执行测试。
