# 预到龄发放通知测试用例矩阵

## 说明

本文档用于指导后续自动化与人工测试设计，当前阶段只完成矩阵拆分，不执行测试。

## 一、Smoke

| 用例编号 | 页面/模块 | 目标 |
| --- | --- | --- |
| NOTICE-UI-01 | 预到龄通知页 | 页面可访问，生成区与查询区可见 |
| NOTICE-UI-02 | 待遇核定页 | 查询区、批量导入入口、列表可见 |
| NOTICE-UI-03 | 核定审核页 | 查询区、批量通过按钮、列表可见 |

## 二、Flows

| 用例编号 | 流程 | 目标 |
| --- | --- | --- |
| NOTICE-FLOW-01 | 生成通知批次 | 正常生成通知主表和明细 |
| NOTICE-FLOW-02 | 通知明细查看 | 按批次查看人员明细 |
| NOTICE-FLOW-03 | 进入待遇核定 | 从通知页跳转到核定页并带入批次号 |
| NOTICE-FLOW-04 | 单条核定录入 | 录入身份证、银行卡、标准、开始年月 |
| NOTICE-FLOW-05 | 上传材料 ZIP | 上传并解压 `用户ID.zip`，生成预览路径 |
| NOTICE-FLOW-06 | 提交复核 | 单条核定进入待复核 |
| NOTICE-FLOW-07 | 批量审核通过 | 复核人按批次批量通过 |
| NOTICE-FLOW-08 | 生成支付计划 | 按通知批次进入支付计划并生成 |
| NOTICE-FLOW-09 | 数据回查 | 批次、明细、核定、支付表状态一致 |

## 三、Exceptions

| 用例编号 | 异常/边界 | 目标 |
| --- | --- | --- |
| NOTICE-EX-01 | 未选通知月份直接生成 | 正确拦截并提示 |
| NOTICE-EX-02 | 导入 Excel 缺少关键字段 | 正确拦截并提示 |
| NOTICE-EX-03 | ZIP 文件名不符合 `用户ID.zip` | 正确拦截并提示 |
| NOTICE-EX-04 | 未上传材料直接提交复核 | 正确拦截并提示 |
| NOTICE-EX-05 | 无批次号批量审核 | 正确拦截或拒绝 |
| NOTICE-EX-06 | 同月重复生成批次 | 返回已存在或幂等结果 |
| NOTICE-EX-07 | 支付计划重复生成 | 不重复落库 |

## 四、Auth

| 用例编号 | 角色 | 目标 |
| --- | --- | --- |
| NOTICE-AUTH-01 | 经办人 | 可生成通知、可录入核定、可上传材料 |
| NOTICE-AUTH-02 | 复核人 | 可查询待复核记录、可批量审核 |
| NOTICE-AUTH-03 | 无关角色 | 不应看到越权入口 |
