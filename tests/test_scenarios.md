## 🎯 场景1: 完整的人员登记到发放流程

### 场景目标
验证从人员登记到最终发放的完整业务流程。

### 涉及角色
- 经办人（jingban1）
- 复核人（fuhe1）
- 审批人（shenpi1）
- 待遇管理员（daiyu1）
- 财务操作员（caiwu1）

### 测试步骤

#### 第1步: 人员登记（经办人）

1. 打开浏览器，访问: http://localhost:80
2. 输入账号: `jingban1`，密码: `admin123`
3. 点击登录
4. 左侧菜单点击"人员信息管理" > "失地居民登记"
5. 点击右上角"新增"按钮
6. 填写表单：
   ```
   姓名: 测试张三
   性别: 男
   身份证号: 110101199001011234
   出生日期: 1990-01-01
   联系电话: 13800138000
   街道办事处: 选择"安次区街道"
   村居委会: 选择"光明社区"
   失地面积: 5.5
   失地时间: 2020-01-01
   银行名称: 中国工商银行
   银行账号: 6222000012340001234
   开户人: 测试张三
   ```
7. 点击"提交"按钮
8. 确认提示"提交成功"
9. 记录生成的ID: _______
10. 退出登录

**预期结果**: ✅ 提交成功，状态为"待复核"

#### 第2步: 人员登记复核（复核人）

1. 使用账号 `fuhe1` 登录
2. 左侧菜单点击"人员信息管理" > "人员登记审核"
3. 在列表中找到"测试张三"的记录
4. 确认状态为"待复核"
5. 点击"详情"按钮，查看信息
6. 点击"通过"按钮
7. 输入复核意见: "信息核实无误，同意通过"
8. 点击"确定"
9. 确认提示"复核成功"
10. 刷新列表，状态变为"待审批"
11. 退出登录

**预期结果**: ✅ 复核通过，状态变为"待审批"

#### 第3步: 人员登记审批（审批人）

1. 使用账号 `shenpi1` 登录
2. 左侧菜单点击"人员信息管理" > "人员登记审核"
3. 在列表中找到"测试张三"的记录
4. 确认状态为"待审批"
5. 点击"详情"按钮，查看信息和审批历史
6. 点击"通过"按钮
7. 输入审批意见: "符合政策要求，批准通过"
8. 点击"确定"
9. 确认提示"审批成功"
10. 刷新列表，状态变为"已通过"
11. 退出登录

**预期结果**: ✅ 审批通过，状态变为"已通过"

#### 第4步: 待遇核定（经办人）

1. 使用账号 `jingban1` 登录
2. 左侧菜单点击"待遇核定管理" > "待遇核定"
3. 在搜索框输入"测试张三"，点击搜索
4. 找到记录，点击"核定"按钮
5. 填写核定信息：
   ```
   补贴类型: 失地居民
   月补贴标准: 1000.00 元
   开始月份: 2026-01
   备注: 按照失地面积5.5亩计算
   ```
6. 点击"提交"按钮
7. 确认提示"核定提交成功"
8. 记录核定ID: _______
9. 退出登录

**预期结果**: ✅ 核定提交成功，状态为"待复核"

#### 第5步: 待遇核定复核和审批（复核人、审批人）

1. 重复第2步和第3步的流程
2. 使用 `fuhe1` 进行复核
3. 使用 `shenpi1` 进行审批
4. 最终状态变为"已通过"

**预期结果**: ✅ 核定审批通过

#### 第6步: 生成支付计划（系统管理员）

1. 使用账号 `admin` 登录
2. 左侧菜单点击"支付计划管理" > "支付计划生成"
3. 选择：
   ```
   补贴类型: 失地居民
   发放月份: 2026-01
   ```
4. 点击"生成计划"按钮
5. 查看生成结果：
   ```
   批次号: BATCH202601190001
   总人数: X 人
   总金额: XXX 元
   ```
6. 记录批次号: _______
7. 退出登录

**预期结果**: ✅ 支付计划生成成功

#### 第7步: 支付计划复核和审批（复核人、审批人）

1. 使用 `fuhe1` 登录，复核支付计划
2. 使用 `shenpi1` 登录，审批支付计划
3. 使用 `shenpi1` 登录，上传到财务系统

**预期结果**: ✅ 支付计划审批通过，已上传财务系统

#### 第8步: 提交银行发放（财务操作员）

1. 使用账号 `caiwu1` 登录
2. 左侧菜单点击"财务管理" > "批次管理"
3. 找到批次号 BATCH202601190001
4. 点击"提交银行"按钮
5. 确认提交
6. 状态变为"已提交银行"

**预期结果**: ✅ 提交银行成功

#### 第9步: 导入发放结果（财务操作员）

1. 左侧菜单点击"财务管理" > "银行发放"
2. 找到批次号 BATCH202601190001
3. 点击"导入发放结果"按钮
4. 上传Excel文件（模拟银行返回的发放结果）
5. 查看导入结果：
   ```
   成功: X 条
   失败: X 条
   ```

**预期结果**: ✅ 导入成功，发放状态已更新

#### 第10步: 查看审计记录（审计查询员）

1. 使用账号 `shenji1` 登录
2. 左侧菜单点击"审计管理" > "审批历史"

1. 使用账号 `daiyu1` 登录
