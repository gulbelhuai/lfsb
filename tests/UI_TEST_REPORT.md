## 四、完整业务流程闭环验证

### 流程 1：人员登记 → 审核通过

```
经办人(operator01) 新增失地居民"陈建国"
  → draft
经办人 提交审核
  → pending_review
复核人(reviewer01) 审核通过
  → approved ✅
```

### 流程 2：人员登记 → 审核驳回

```
经办人(operator01) 新增失地居民"刘美华"
  → draft
经办人 提交审核
  → pending_review
复核人(reviewer01) 驳回（身份证号与户籍信息不一致）
  → rejected ✅
```

### 流程 3：待遇核定 → 审核通过

```
经办人(operator01) 输入身份证号搜索 → 自动关联"陈建国"
  → 选择失地居民、月标准500、2026-03、工商银行
  → draft
经办人 提交核定
  → pending_review
复核人(reviewer01) 审核通过
  → approved ✅
```

### 流程 4：补贴发放 → 审核 → 发放

```
经办人(operator01) 搜索"陈建国" → 选择失地居民补贴
  → 创建发放记录 → 待审核(1)
管理员(admin) 审核通过
  → 待发放(2)
管理员(admin) 确认发放
  → 已发放(4) ✅
```
