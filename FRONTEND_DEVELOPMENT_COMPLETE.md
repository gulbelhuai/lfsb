## 📌 重要提示

### 首次运行

1. **安装依赖**
```bash
cd ruoyi-ui
npm install
```

2. **配置API地址**
```bash
# .env.development
VUE_APP_BASE_API = 'http://localhost:8080'
```

3. **启动前端**
```bash
npm run dev
```

4. **访问系统**
```
http://localhost:80
```
